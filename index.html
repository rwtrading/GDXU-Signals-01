TradingView Webhook Setup

To connect TradingView alerts, use this endpoint as your webhook URL:

https://your-deployed-url.com/webhook

Step-by-step setup in TradingView:

1. Open each indicator (DXY, TNX, XAUUSD, WTI, VIX, GDX) on TradingView

2. Create an Alert → set condition → enable Webhook URL

3. In the message body, use this JSON template:

{
  "symbol": "{{ticker}}",
  "value": {{close}},
  "change_pct": {{change}},
  "timestamp": "{{time}}"
}

Symbols to set up alerts for:

DXY · US10Y (TNX) · XAUUSD · USOIL · VIX · TLT · XAGUSD · GDX
